1. Install [pipx](https://pipx.pypa.io/stable/installation/)

2. Use a Github template

```bash
pipx run cookiecutter gh:ThinkR-open/talospy
```

3. Answer the questions

4. Use poetry shell

```bash
cd talospy
poetry shell
```

5. Install the dependencies

```bash
poetry install
```

6. Run the application

```bash
poe runshiny
```
